Welcome to proto-poll project's home page

Proto-poll is an in-development polling system releases under the free licence GPL3.

I believe builing such a software by my own will only lead to a POF (Proof Of Failure). Therefore, if you're interested in this project, any help is welcome:

  • users to test the application, define useful usecases;
  • designers to make it nicer than it is;
  • developers to implement what users need (let's get a very small time to market!);
  • you... to do what you are good at!
